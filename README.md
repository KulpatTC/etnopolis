# Этнополис

Веб-сайт о культурном и этническом разнообразии России. Итоговый проект Яндекс Лицея.

## О проекте

Etnopolis — информационный сайт, посвящённый народам, культурам и традициям России. Сайт позволяет узнать о народах страны, их традициях и особенностях культуры, а также следить за актуальными этнокультурными событиями по всей России.

## Страницы

- **Главная** — обзор проекта
- **Народы** — карточки народов России с описанием традиций и культуры
- **События** — этнокультурные фестивали и мероприятия 2026 года
- **Викторина** — тест на знание культур народов России

## REST API

Базовый URL: `/api`

| Эндпоинт | Описание |
|---|---|
| `GET /api/nations` | Все народы |
| `GET /api/nations/<id>` | Народ по ID |
| `GET /api/nations/<name>` | Народ по названию |
| `GET /api/events` | Все события |
| `GET /api/events/<city>` | События по городу |
| `GET /api/tasks` | Все вопросы викторины |
| `GET /api/tasks/<id>` | Вопрос по ID |
| `GET /api/tags` | Все теги |
| `GET /api/tags/<id>` | Тег по ID |

## Стек

- **Backend:** Python, Flask, SQLAlchemy
- **Frontend:** HTML, CSS, Jinja2
- **БД:** SQLite
- **Данные:** JSON (народы, события)

## Запуск

```bash
pip install -r requirements.txt
python server.py
```

Сайт будет доступен на `https://localhost:80/`
